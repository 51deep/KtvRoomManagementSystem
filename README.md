## 功能介绍（完整见运行截图）
管理员：登录、注册、注销管理，系统主界面导航和功能模块管理，包厢信息和状态管理，用户和会员信息管理，商品类型和采购管理，包厢预约与审核管理，留言反馈查看与回复，账单信息管理与财务核算，发布活动与优惠资讯，上传与管理系统相关图片和文件。用户：通过注册登录系统，访问个人中心查看和更新个人信息，浏览并预订包厢，查看商品列表并购买商品，提交留言反馈，接收及使用特定折扣与优惠信息。
